# Modelo de Predicción de Lesiones

## Algoritmo
- Random Forest o Árbol de Decisión

## Variables de Entrada
- Edad
- Peso
- Altura
- Posición
- Minutos jugados
- Lesiones previas

## Salida
- Probabilidad de lesionarse en los próximos 3 meses (%)
- Número estimado de lesiones en la temporada actual

## Herramientas
- Python 3.x
- scikit-learn
- pandas
- joblib para persistencia del modelo
