# Roadmap de La Curva del Banquillo

## Fase 1 (0-30 días)
- Crear README y documentación inicial.
- Configurar entorno local (PHP, MySQL, Python).

## Fase 2 (1-3 meses)
- Implementar backend básico y endpoints.
- Crear estructura de DB y migraciones.
- Desarrollar frontend inicial.

## Fase 3 (3-6 meses)
- Integrar ML para predicción de lesiones.
- Integrar Llama2 para preguntas dinámicas.

## Fase 4 (6-12 meses)
- Optimización de UX/UI.
- Tablas comparativas de jugadores.
- Automatización de datos y trivias.
